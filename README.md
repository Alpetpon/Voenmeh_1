# Генератор случайных чисел

Веб-приложение для генерации случайных чисел с возможностью настройки диапазона.

## Функционал

- **Генерация случайных чисел** - клик в любом месте экрана генерирует новое число
- **Настройка диапазона** - отдельная страница для изменения минимального и максимального значений
- **Сохранение настроек** - настройки сохраняются в сессии пользователя
- **Валидация** - проверка корректности введенных значений

## Технологии

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Контейнеризация**: Docker

## Запуск

### Локальный запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер: http://localhost:5000

### Docker

1. Соберите образ:
```bash
docker build -t voenmeh-randomizer .
```

2. Запустите контейнер:
```bash
docker run -p 5000:5000 voenmeh-randomizer
```

Подробные инструкции по Docker: [docker-instructions.md](docker-instructions.md)

## API

- `GET /` - главная страница
- `GET /settings` - страница настроек
- `POST /settings` - сохранение настроек
- `POST /generate` - генерация случайного числа (AJAX)

## Особенности

- Настройки сохраняются в сессии пользователя
- По умолчанию диапазон: 1-100
- Валидация: минимальное число должно быть меньше максимального
- Адаптивный дизайн
- Современный UI с анимациями
